## Cortlandt

Scraper of rental flats on offer in specific areas. 


### Requirements

  * pygsheets and a Google Sheet setup
  * Key to Google Maps API
  * Docker - I run this as a container
  * googlemaps, pysheets, selenium from pip
  * phantomjs
